**DGGang** is a Stardew Valley mod developed speficially for our friend group, adding immersion to the gameplay experience by referencing real life lore of our players!

## Contents
- [Install](#install)

## Install
1. [Install the latest vesion of SMAPI](https://smapi.io/)
2. Navigate to the [Releases](https://github.com/bhan-trinh/DGGang-SDV-Pack/releases) page of the mod
3. Download the source code as a `zip/tar.gz` and unzip into your `Mods` folder
4. Have fun!
